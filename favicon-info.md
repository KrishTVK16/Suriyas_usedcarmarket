# Favicon Setup Instructions

To update the favicon for UsedCar Market:

## Required Favicon Files

Place these files in the `public` folder:

1. **favicon.ico** - Main favicon (16x16, 32x32, 48x48 sizes)
2. **favicon-16x16.png** - 16x16 PNG favicon
3. **favicon-32x32.png** - 32x32 PNG favicon
4. **apple-touch-icon.png** - 180x180 PNG for iOS devices
5. **logo192.png** - 192x192 PNG (already exists, update if needed)
6. **logo512.png** - 512x512 PNG (already exists, update if needed)

## Design Guidelines

- Use the UsedCar Market branding colors:
  - Primary: Redline Red (#FF2B2B)
  - Background: Midnight Black (#101010)
- Include car-related iconography or the "UC" initials
- Ensure good visibility on both light and dark backgrounds
- Keep design simple and recognizable at small sizes

## Online Favicon Generators

You can use these tools to generate favicons:
- https://favicon.io/
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

## Current Configuration

The HTML and manifest.json files are already configured to use these favicon files.
Once you add the image files, they will automatically be used by the browser.
